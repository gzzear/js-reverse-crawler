# 易语言调用 PDD Anti-Content API 示例

本文档演示如何在易语言中调用 PDD Anti-Content API 接口。

## 前提条件

1. Express 服务已启动（运行 `./start.sh` 或 `node server.js`）
2. 服务运行在 `http://localhost:3000`
3. 易语言已安装 HTTP 访问相关支持库（如：精易模块、网页访问支持库等）

## 重要说明

- **只支持 GET 请求**：接口简单易用，直接访问 URL 即可
- **自动时间戳**：服务自动使用当前时间生成 `anti_content`，无需传入任何参数
- **动态生成**：每次调用都会生成不同的结果

## JSON 解析说明

如果你的易语言没有 JSON 解析库，可以使用以下简单方法：

```易语言
如果结束

如果 (结束位置 = -1)
    返回 ("")
如果结束

返回 (取文本中间 (JSON文本, 开始位置, 结束位置 - 开始位置))
```

## 常见问题

### Q1: 提示连接失败？
**A:** 请确保 Express 服务已启动，可以在浏览器访问 `http://localhost:3000/health` 测试。

### Q2: 返回空或错误？
**A:** 检查服务日志文件 `server.log`，看是否有错误信息。

### Q3: 如何更改服务器地址？
**A:** 如果服务运行在其他地址或端口，修改代码中的 URL：
```易语言
返回内容 = 网页_访问("http://192.168.1.100:8080/api/anti_content", 0, , , , , )
```

### Q4: JSON 解析失败？
**A:** 可以使用简单的文本提取方法，或者安装精易模块等第三方库。

## 推荐的易语言 JSON 库

1. **精易模块** - 功能强大，使用简单
2. **火山互联** - 官方支持，性能好
3. **E2EE** - 轻量级，易上手

## 测试建议

1. 先用浏览器测试接口是否正常
2. 使用 Postman 或 curl 验证 API
3. 在易语言中逐步调试，先输出原始响应
4. 确认 JSON 解析库正常工作

## 技术支持

如有问题，请：
1. 检查服务器日志：`tail -f server.log`
2. 测试健康检查接口
3. 验证网络连接是否正常
